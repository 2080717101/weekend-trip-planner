# 🗺️ 周末出游规划助手

一个基于高德地图的周末出游规划工具，帮助您根据家庭需求智能推荐目的地和路线。

## ✨ 功能特点

### 输入模块
- 📊 **出游人数**：支持1-10人
- 🥾 **地形偏好**：平路友好 / 不限地形
- 📅 **出行类型**：当天往返 / 多日游
- 🏠 **出发地点**：
  - 您的家（昌平区龙华园一区）
  - 父母家（西城区广内大街200号楼）
  - 自定义地址
- 🚗 **驾车距离**：30/50/100/200公里范围

### 推荐引擎
- 基于位置智能推荐北京及周边景点
- 考虑到两家的距离平衡
- 根据人数、地形、出行类型筛选
- 提供游玩路线和停车场信息

### 可视化展示
- 🗺️ 高德地图实时显示
- 📍 起点、终点、沿途景点标记
- 🚗 自驾路线（含预计时间和距离）
- 🚶 游玩路线建议

### 分享功能
- 生成分享链接
- 复制文本方案
- 支持参数化分享

## 🚀 使用方法

### 本地使用
1. 双击 `index.html` 文件
2. 浏览器会自动打开网页
3. 填写需求，点击「生成出游方案」
4. 查看地图和推荐路线

### 分享给朋友
1. 点击「生成出游方案」
2. 点击「📤 分享给朋友」
3. 复制链接或文本
4. 发送给您的朋友

## 🔧 技术栈

- **前端**：纯HTML/CSS/JavaScript
- **地图**：高德地图 JS API 2.0
- **部署**：静态文件，无需后端

## 🔄 数据更新

如需添加新的目的地，编辑 `script.js` 文件中的 `DESTINATIONS` 数组。

每个目的地包含：
- name: 名称
- address: 地址
- lat/lng: 坐标
- distance: 到市中心的距离（公里）
- terrain: 地形（flat/any）
- suitableFor: 适合的人数
- highlights: 亮点介绍
- parking: 推荐停车场
- walkRoute: 游玩路线
